Local Long Tail Creates Multiple Location Based SEO Optimized Posts

== Description ==

Local Long Tail Creates Multiple Location Based SEO Optimized Posts

== Installation ==

1. Install Local Long Tail either via the WordPress plugin directory or by uploading the files to your server at wp-content/plugins.

2. After activating Local Long Tail, navigate to the settings page at "Local Long Tail" Page to configure the link distribution service.

3. Use the link at the top of the page to redirect traffic.

== How to use ==

Install and activate the Local Long Tail plugin.

Creating Child Posts

1. Creating a Location:
    a. On the left menu, hover over 'Local Long Tail' menu item.
    b. On the drop down, select  'Location Groups'.
    c. After the page loads, click on 'New Group'.
    d. Enter the location name and press 'Save Location Group'.
    e. Congratulations, you have created a location group.
    f. Close "Location Groups" tab and click on the new Location group.
    g. Click "New Child Location" add new child location.


2. Create Parent Post
    a. On the left menu, hover over 'Local long Tail' menu item.
    b. On the drop down, select  'Parent Post'
    c. After the page loads, click on 'New Parent Post'.
    d. After the page loads, edit the 'Title' and 'Content' following.
    e. At the bottom, click 'Save'.
    f. Navigate to the top to edit keywords. Click on "Manage Keywords and Location".
    g. Add a keyword.
    h. Select locations for the keyword.
    i. At the top menu, select "Generate Children Posts".
    k. Click "Generate Posts".
    l. Congratulations you have successfully generated.
